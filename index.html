Manual Prerequisites

Three essential tools that need manual installation before we automate the rest.

1

Xcode Command Line Tools

Apple's development tools that everything else depends on. This includes compilers, headers, and other build essentials.

xcode-select --install

‚úì Click "Install" in the popup window

‚úì Accept the license agreement

‚úì Wait ~10 minutes for download and installation

Verify it worked: 
xcode-select -p
Should output: /Library/Developer/CommandLineTools
2

Homebrew Package Manager

The missing package manager for macOS. We'll use this to install everything else.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

During installation:

  • You'll be prompted for your Mac password
  • Press Enter when asked to confirm
  • Installation takes 2-5 minutes
‚ö†Ô∏è Apple Silicon Macs: After installation, you need to add Homebrew to your PATH: 
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc
Verify installation: 
brew --version
3

iTerm2 Terminal Emulator

A powerful replacement for Terminal.app with better features for power users.

brew install --cask iterm2

Recommended iTerm2 Configuration

After installation, open iTerm2 and configure these settings:

1. General Settings (‚åò,)
  • Appearance ‚Üí Theme: Minimal (cleaner look)
  • Profiles ‚Üí Terminal: Unlimited scrollback
  • Profiles ‚Üí Keys ‚Üí Key Mappings: Presets ‚Üí Natural Text Editing
2. Color Scheme
  • Go to Profiles ‚Üí Colors
  • Click Color Presets... dropdown
  • Try: Solarized Dark or Dracula
3. Font Settings
  • Go to Profiles ‚Üí Text
  • Font: JetBrains Mono or SF Mono
  • Size: 13-14pt for readability
  • Enable: Use ligatures (if using JetBrains Mono)
4. Window Management
  • Window ‚Üí Save Window Arrangement: Save your layout
  • General ‚Üí Startup: Open saved arrangement
  • Keys: ‚åòD (split vertical), ‚åò‚áßD (split horizontal)
üí° Quick Tip: Install JetBrains Mono font for better code readability: brew tap homebrew/cask-fonts && brew install --cask font-jetbrains-mono

Automated Requirements Installation

Now that prerequisites are done, install everything else with a single command.

üöÄ One-Line Install

Copy and paste this command to install all NetOps tools and configurations:

curl -fsSL https://brndnsvr.github.io/cnsq-mac-setup/install.sh | bash

What This Installs:

Development Tools

Git, Neovim, tmux, Python, Node.js, jq, ripgrep

Network Tools

nmap, mtr, telnet, netcat, dig, sipcalc

Automation

Ansible + collections, Python libraries, expect

Shell Enhancements

eza, syntax highlighting, GitHub CLI, custom functions

What Happens:

  1. Checks for Homebrew (exits if not found)
  2. Installs all tools via brew
  3. Creates Python virtual environment at ~/.cnsq-venv
  4. Installs Python packages (netmiko, napalm, paramiko, etc.)
  5. Installs Ansible collections (Cisco, Juniper, Arista)
  6. Configures shell with aliases and functions
Alternative: Download and review the script first: 
curl -O https://brndnsvr.github.io/cnsq-mac-setup/install.sh
cat install.sh  # Review the script
bash install.sh # Run it

Core Tools

Essential tools that form the foundation of your development environment.

üç∫

Homebrew

The missing package manager for macOS. Install and manage software from the command line.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Common Uses:

  • brew install git - Install packages
  • brew update - Update package lists
  • brew upgrade - Upgrade installed packages
Quick Start: After installation, run brew doctor to verify everything is working.
üå≥

Git

Distributed version control system for tracking changes in code.

brew install git

Configuration:

git config --global user.name "Your Name"
git config --global user.email "your.email@company.com"
Quick Start: Clone your first repo with git clone git@github.com:org/repo.git
üìù

Neovim

Hyperextensible text editor. A modern fork of Vim with better defaults and Lua scripting.

brew install neovim

Why Neovim for NetOps?

  • Edit configs directly on remote systems
  • Powerful search/replace with regex
  • Works over SSH without GUI
Quick Start: Create an alias: alias vi='nvim' in your shell config
üñ•Ô∏è

tmux

Terminal multiplexer. Run multiple terminal sessions in a single window.

brew install tmux

Essential Commands:

  • tmux new -s work - New named session
  • tmux attach -t work - Reattach to session
  • Ctrl+b % - Split vertically
  • Ctrl+b " - Split horizontally
NetOps Use: Keep long-running network scans alive even if SSH disconnects

Combines ping and traceroute for network diagnostics.

# Report mode (10 cycles)

Automation Example:

    sshpass -p "$PASS" ssh user@$ip "show version"

Automation & Configuration Management

Tools for automating network configuration and management at scale.

Ansible

Agentless automation platform for network configuration management.

brew install ansible

Install Network Collections:

ansible-galaxy collection install cisco.ios
ansible-galaxy collection install cisco.iosxr
ansible-galaxy collection install junipernetworks.junos
ansible-galaxy collection install arista.eos

Example Playbook:

---
- name: Backup Network Configs
  hosts: network_devices
  gather_facts: no
  tasks:
    - name: Backup IOS config
      cisco.ios.ios_config:
        backup: yes
        backup_options:
          filename: "{{ inventory_hostname }}.cfg"
Getting Started: Create ~/.ansible.cfg with: 
[defaults]
host_key_checking = False
timeout = 30
[persistent_connection]
command_timeout = 60

Python Network Libraries

Essential Python packages for network automation.

Core Libraries:

pip install netmiko    # SSH connections to network devices
pip install napalm     # Multi-vendor network automation
pip install paramiko   # SSH2 protocol library
pip install textfsm    # Parse semi-structured text
pip install pyyaml     # YAML parser for configs
pip install jinja2     # Template engine

Quick Example - Netmiko:

from netmiko import ConnectHandler

device = {
    'device_type': 'cisco_ios',
    'host': '192.168.1.1',
    'username': 'admin',
    'password': 'password'
}

with ConnectHandler(**device) as net_connect:
    output = net_connect.send_command('show ip int brief')
    print(output)

Shell Functions

Custom functions to speed up common tasks.

Add to ~/.zsh/functions.zsh:

# Quick SSH to device
sshto() {
    ssh admin@"$1"
}

# Ping sweep
pingsweep() {
    for i in {1..254}; do
        ping -c 1 -W 1 192.168.1.$i &>/dev/null && echo "192.168.1.$i is up"
    done
}

# Extract any archive
extract() {
    case $1 in
        *.tar.gz)  tar xzf "$1" ;;
        *.tar.bz2) tar xjf "$1" ;;
        *.zip)     unzip "$1" ;;
        *)         echo "Unknown archive type" ;;
    esac
}
